Mac Adventures


Mac Adventures is a program held by McDonalds in Japan. The program allows kids to have hands on expirence with making burgers and other foods. Later, the program was additionally put into a game in collaberation with GOKKO LAND. The game version was released inside of an app named "GEKKO LAND" (known as "pretendland" in the U.S.). This game is not a continuation to DS Mac Adventures but does seem to reuse some sound effects from eCDP.
