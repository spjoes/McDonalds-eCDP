DS Mac Adventure (DS マック アドベンチャー)


DS Mac Adventure (DS マック アドベンチャー) was a game developed based on eSmart 2.0. The game was watered down from eSmart 2.0 to better reach their target audience (being children). The game was able to be played and downloaded for a limited time in McDonalds across japan through the "Mac De Ds (マックでDS)" Program and through a "Parents and Children's Day" event held several times throughout 2011 but the game deleted itself off the DS once the DS was powered off. The purpose of the game was to train children who just might be, the next generation of McDonalds employee's

Not much is known about this game so its hard to give much info on it.

If you would like to read the official documentation on the game by McDonalds Japan that can be found here (please note that this website is written in Japanese).
