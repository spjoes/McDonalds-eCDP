DS Mac Adventure


Not everything is known about this game due to this game being a Nintendo Zone download meaning it required players to go to a McDonalds to download the game and were only able to play the game for a short period of time. Ontop of being a Nintendo Zone exclusive this game was only available in Japan. So far the games files/rom has not been dumped on the internet

If you would like to read the official documentation on the game by McDonalds Japan that can be found here (please note that this website is written in Japanese).
