DS Mac Adventure


DS Mac Adventure was a game developed based on eSmart 2.0. The game was able to be played and downloaded for a limited time in McDonalds across japan through the "Mac De Ds" Program and through a "Parents and Children's Day" event held several times throughout 2011.

Not much is known about this game so its hard to give much info on it.

If you would like to read the official documentation on the game by McDonalds Japan that can be found here (please note that this website is written in Japanese).
