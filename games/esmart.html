eSmart 2.0


Not much is known about this game but it seems to be a version of the eCDP game. eSmart seems to have been created before eCDP and was used to train workers that were already working at McDonalds and therefore was more challenging than eCDP.

so far there are 2 known versions of this game. eSmart & eSmart 2.0. There were known auctions on this game but the outcomes are still unknown so the game remains a mystery, but as of 12/4/2020 (December 4th, 2020) there is one known person that is believed to be in possesion of an eSmart cartridge but the status of this is still undetermined
